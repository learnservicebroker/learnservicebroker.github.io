Content Sources

Content Sources provide the templates that become catalog items in the Service Broker Catalog .

Content Source Types

The following Content Source types can be configured:

  • Extensibility Actions - configure Cloud Assembly Action Based Extensibility (ABX) actions as Catalog Items
  • vRealize Orchestrator Workflow - configure vRealize Orchestrator Workflows as Catalog Items (requires a vRealize Orchestrator integration in Cloud Assembly )
  • AWS CloudFormation Template - configure AWS CloudFormation templates as Catalog Items
  • Marketplace VM Templates (OVA) - configure VMware (Bitnami) Marketplace OVAs as Catalog Items
  • Code Stream Pipeline - configure Code Stream Pipelines as Catalog Items
  • VMware Cloud Templates - configure Cloud Assembly Cloud Templates as Catalog Items
  • vRA 7.x Catalog Items - BETA configure legacy vRealize Automation 7.x catalog items as Catalog Items (requires a vRA 7.x Integration in Cloud Assembly )
  • vRealize Automation 8.x Template - (vRA Cloud only) configure vRealize Automation 8.x on-premises Cloud Templates as a Catalog Item (requires vRealize Cloud Universal)
